四.标准文档流

1.块级元素和行内元素

1） 块级元素 div、h、p、 li、dt、dd

  • 霸占一行，不能与其他任何元素并列
  • 能接受宽、高
  • 如果不设置宽度，那么宽度将默认变为父亲的100%。

2） 行内元素 p、span、a、b、i、u、em

  • 与其他行内元素并排
  • 不能设置宽、高。默认的宽度，就是文字的宽度。
