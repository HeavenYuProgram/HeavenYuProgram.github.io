一.CSS 层叠式样式表

  • p h1 选择器
  • key value 对
  • 对空格换行不敏感
  • 格式要对
  • CSS负责样式,HTML负责结构,JS负责行为.

二.选择器

1.标签选择器

  • 选择的是全部 描述共性

2.id选择器 #id

  • 任何的HTML标签都可以有id属性。表示这个标签的名字。
  • 只能有字母、数字、下划线
  • 必须以字母开头
  • 不能和标签同名。比如id不能叫做body、img、a
  • 一个HTML页面，不能出现相同的id
  • 一个标签可以被多个css选择器选择，共同作用，这就是“层叠式”的第一层含义。
  • css有着非常严格的计算公式，能够处理冲突

3.类选择器 点.

  • 既能选择部分,又不像id要每个起不同的名字

4.正确使用类 每个类定义一种特性

  • CSS尽量用class id是js用的
  • 类上样式,id上行为
  • 所以有id的一般有动态效果

三.高级选择器

1.后代选择器 空格 .div1 .li2 p

也可以用标签后代

  • 只要满足祖先结构就可以
  • Chrome可以看祖先结构

2.交集选择器 h3.special 紧连

  • 紧连的是取交集 也就是后边类的标签
  • 一般以标签开头

3.并集选择器 逗号, (分组选择器)

  • *表示全部 通配符

4.CSS3选择器

1.儿子选择器 >

2.序选择器 :-child

  • :first-child

  • :last-child

  • :nth-child(2n+2)

3.下一个兄弟选择器 +

四.继承性 有些属性会继承给后代

  • color text开头的 line开头的 font开头的 文字样式的可以继承
  • 关于盒子的 定位的 布局的属性不能继承

五.层叠性

  • 处理冲突的能力
  • 都直接选中某个元素, 权重=id的数量，类的数量，标签的数量 依次选大的
  • 如果权重一样，那么以后出现的为准
  • 如果不能直接选中某个元素，通过继承性影响的话，那么权重是0。
  • 如果大家都是0，那么有一个就近原则：谁描述的近，听谁的。

六.兼容性

  • 下列都是IE6兼容的选择器：

p

#box

.spec

div.box

div .box

div , .box

  • 下列都是IE7开始兼容：

div>p

div+p

  • 下列都是IE8开始兼容：

div p:first-child

div p:last-child
